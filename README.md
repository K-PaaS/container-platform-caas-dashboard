## 시작하기

PaaS-TA 에서 제공하는 컨테이너 서비스 사용을 쉽게 하기 위하여 제공하는 웹 애플리케이션입니다.


### 빌드 방법

PaaS-TA 컨테이너 서비스 DASHBOARD 소스 코드를 활용하여 로컬 환경에서 빌드가 필요한 경우 다음 명령어를 입력합니다.
```
$ gradle build
```
